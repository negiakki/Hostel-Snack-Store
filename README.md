\# Hostel Snack Store



A lightweight web application for running a hostel-based night snack store.



The application helps a student manage products, inventory, customer orders, and business analytics from a single dashboard, eliminating the need for spreadsheets or manual calculations.



\---



\# Overview



Hostel Snack Store is \*\*not\*\* a full e-commerce platform or warehouse management system.



It is intentionally designed for \*\*one student running a small snack store from their hostel room\*\* during a few hours each night.



The goal is simple:



> Make running tonight's shop as easy as possible.



\---



\# Key Features



\* 🏪 Store open/close management

\* ⏰ Automatic operating schedule with manual override

\* 🛍️ Product management

\* 📦 Inventory tracking

\* 📋 Order management

\* 📊 Automatic revenue and profit analytics

\* 📱 Responsive admin dashboard



\---



\# Product Philosophy



Every feature should answer one question:



> \*\*"Would this genuinely make running tonight's shop easier?"\*\*



If the answer is \*\*no\*\*, the feature belongs in a future version or should be removed.



Core principles:



\* Keep Version 1 simple.

\* Automate repetitive work.

\* Preserve historical accuracy.

\* Avoid unnecessary complexity.

\# Typical Workflow



Every night, the shopkeeper:



1\. Opens the store.

2\. Receives customer orders.

3\. Prepares orders.

4\. Delivers orders.

5\. Closes the store.

6\. Reviews analytics.
